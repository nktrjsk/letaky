# Letáky

Tisknutelné A5 letáky projektu [Samostuduj](https://samostuduj.cz) — Markdown
zdroje publikované z [editoru letáků](https://github.com/nktrjsk/flyer-editor).
Každý leták žije ve `flyers/<slug>/` jako verzované soubory `v1.md`, `v2.md`, …
s případným logem.

Web [samostuduj.cz/letaky](https://samostuduj.cz/letaky) si tento repozitář
stahuje při buildu a generuje z něj PDF ke stažení i náhledy; push do `main`
web automaticky přebuildí (viz `.github/workflows/samostuduj-deploy.yml`).
